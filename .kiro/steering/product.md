# Product Overview

PerfMon is a Lighthouse performance monitoring tool that provides instant web performance audits with historical tracking. It analyzes websites for performance, accessibility, best practices, and SEO metrics.

## Key Features

- Run Lighthouse audits on any URL (local or remote)
- Support for both mobile and desktop device emulation
- Local history tracking using IndexedDB (no backend database required)
- Visual screenshot thumbnails of analyzed pages
- Detailed audit breakdowns showing specific issues and recommendations
- Dark mode support with system preference detection
- Optional Tauri desktop app wrapper for native experience

## Target Users

Developers who need quick, repeatable performance audits during development and want to track improvements over time without complex setup.
